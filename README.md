# Hotspoto

An easy and simple Wi-Fi hotspot app for Linux users who has single Wi-Fi adapter.

WARNING: Only works for Fedora Linux, if you want other distros please open a issue in this repo.

### Dependencies

- Python (Probably already installed)
- create_ap (Probably already installed)

### Installation and Usage

```
git clone https://www.github.com/detherminal/Hotspoto

cd Hotspoto

sudo python main.py
```
